FriendZone

FriendZone

Fall 2021

FriendZone is a proximity based friend finding app created to connect like-minded people in a new environment, such as going off to college. Users create a profile based on their interests and the app facilitates face to face meetings with other nearby users who share similar interests.

Project Contributions

  • Performed user research by conducting interviews with a target group to identify user needs
  • Created affinity diagrams using Miro to identidy user needs
  • Created paper prototypes of based on initial UI sketches
  • Designed hi-fidelity UI wireframe in Figma
  • Performed heuristic evaluation to improve usability of wireframe
